Atelier Pratique – Classification avec Arbre de Décision

Dataset Santé

Dans cet atelier, nous allons entraîner un arbre de décision pour prédire l’état de santé général (genhlth) d'une personne en fonction de plusieurs facteurs :

  • exerany : pratique du sport
  • hlthplan : couverture santé
  • smoke100 : a fumé plus de 100 cigarettes
  • height : taille (inches/cm selon dataset)
  • weight : poids
  • age : âge
  • gender : sexe (0 = Femme, 1 = Homme)

Le modèle entraîné est sauvegardé et pourra être utilisé lorsque l’utilisateur saisira ses propres données dans un formulaire, puis cliquera sur "Tester le modèle".
