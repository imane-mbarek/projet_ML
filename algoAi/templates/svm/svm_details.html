Algorithme : SVM (Support Vector Machine)

Détails :

Le Support Vector Machine (SVM) est un algorithme d’apprentissage supervisé utilisé principalement pour la classification, mais aussi pour la régression (SVR). Il cherche à trouver l’hyperplan optimal qui sépare les classes avec la plus grande marge possible.

Caractéristiques :

  • Très performant pour les données à forte dimension
  • Utilise des noyaux (kernel) : linéaire, RBF, polynomial…
  • Efficace pour les séparations non linéaires
  • Robuste face aux valeurs extrêmes
