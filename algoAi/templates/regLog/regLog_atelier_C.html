Atelier Pratique – Classification des véhicules

Dataset des véhicules

L’objectif de cette démonstration est de vous montrer comment on entraîne un modèle de régression logistique pour classifier les véhicules en Camion ou Touristique selon le nombre de roues et la hauteur donnée. Les modèles entraînés sont sauvegardés de cette atelier et vont être utilisés pour faire des classifications à base des données qui vont être saisir par l’utilisateur de l’application via un formulaire une fois il clique sur le bouton "Tester le modèle".
